# viewer

ローカルに保存された画像・動画ファイルを高速・快適に閲覧・整理するデスクトップアプリケーション。

## 特徴

- **高速なサムネイル表示** — Rust バックエンドによる非同期サムネイル生成とディスクキャッシュ
- **大量ファイル対応** — 仮想スクロールで数千枚のファイルでも快適に動作
- **フィルムストリップビューア** — 画像が縦長の場合、左右に前後の画像を半透明で表示
- **Ken Burns スライドショー** — ズーム＋パンアニメーション付きのスライドショー
- **パララックスビューア** — AI深度推定（WebGPU加速）とWebGLによるマウス連動の視差効果
- **キーボード操作完結** — マウスなしで主要操作がすべて完結

## 対応ファイル形式

| 種別 | 拡張子 |
|------|--------|
| 画像 | jpg / jpeg / png / webp / bmp / tiff / gif / heic / heif |
| 動画 | mp4 / mkv / avi / mov / wmv / flv / webm / m4v |

## 開発環境のセットアップ

### 必要なもの

- [Rust](https://rustup.rs/)（stable チャンネル）
- [Node.js](https://nodejs.org/) v20 以上（nvm 推奨）
- [Tauri の前提条件](https://tauri.app/start/prerequisites/)（Windows: WebView2、Linux: webkit2gtk など）

### インストール

```bash
# Node.js 依存関係のインストール
npm install

# 開発サーバー起動（ホットリロードあり）
npm run tauri dev
```

### ビルド

```bash
# リリースビルド

```

### その他のコマンド

```bash
npm run dev        # Vite 開発サーバーのみ起動

npm run test       # テスト実行（vitest）

```

## 主な機能

### ギャラリー
- フォルダツリーからフォルダを選択してファイルを一覧表示
- サムネイルサイズをスライダーで調整（60〜300px）
- ソート（名前 / 日付 / サイズ / 種別）— 設定は次回起動時も保持

### フィルター
- ファイル名の部分一致検索
- 拡張子単位でのトグル絞り込み
- 日付・サイズの範囲指定
- タグによる AND 検索

### 画像ビューア
- フィルムストリップ表示（前後画像を両端に表示）
- ズーム・回転・反転
- フルスクリーン表示
- `←` / `→` キーでファイル移動（フィルター範囲内）

### スライドショー
- Ken Burns エフェクト（ズーム＋パン）
- クロスフェード遷移
- 切り替え間隔 1〜60 秒で設定可能（デフォルト 10 秒）
- 切り替え待機中に次の画像の深度マップをバックグラウンドで事前計算

### パララックスビューア
- 画像ビューアの 🌊 ボタンで起動
- AI（`onnx-community/depth-anything-v2-small`）で深度マップを推定
- WebGPU 加速推論（非対応環境は WASM に自動フォールバック）
- マウス移動に連動した WebGL 視差効果（標準 / 波モード）

### プレイリスト
- チェックしたファイルの組み合わせを名前付きで保存・読み込み

### タグ管理
- ファイルに任意のタグを付与（SQLite で永続保存）
- 複数ファイルへの一括タグ付け

## キーボードショートカット

### ギャラリー

| キー | 動作 |
|------|------|
| `←` / `→` | 前 / 次のファイルを選択 |
| `Enter` | 選択ファイルを開く |
| `Ctrl+F` | 検索バーにフォーカス |
| `Space` | スライドショー開始 / 停止 |
| `F` | フルスクリーン切替 |
| `I` | 情報パネル切替 |
| `Esc` | スキャン中断 / フルスクリーン解除 |

### 画像ビューア

| キー | 動作 |
|------|------|
| `←` / `→` | 前 / 次の画像 |
| `+` / `-` | ズームイン / アウト |
| `0` | ズームリセット |
| `R` / `Shift+R` | 右 / 左に 90° 回転 |
| `H` / `V` | 左右 / 上下反転 |
| `F` | フルスクリーン切替 |
| `Esc` | 閉じる |

### 動画プレーヤー

| キー | 動作 |
|------|------|
| `Space` | 再生 / 一時停止 |
| `←` / `→` | 5 秒戻し / 5 秒送り |
| `↑` / `↓` | 音量 +10% / -10% |
| `M` | ミュート切替 |
| `L` | ループ切替 |

