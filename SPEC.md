# viewer — 機能仕様書

**バージョン**: 0.1.0
**最終更新**: 2026-03-16
**対象OS**: Windows 10 / 11

---

## 1. 概要

`viewer` はローカルに保存された画像・動画ファイルを高速・快適に閲覧・整理するためのデスクトップアプリケーションです。

### 技術スタック

| レイヤー | 技術 |
|----------|------|
| フレームワーク | Tauri 2 |
| バックエンド | Rust |
| フロントエンド | React 18 + TypeScript |
| スタイリング | Tailwind CSS |
| 状態管理 | Zustand |
| 仮想スクロール | TanStack Virtual |
| DB | SQLite（rusqlite） |

---

## 2. レイアウト構成

```
┌─────────────┬──────────────────────────────┬──────────┐
│  左サイドバー  │        メインエリア           │ 右パネル  │
│  (幅 224px)  │                              │ (幅 256px│
│             │  ┌─ 検索バー ─────────────┐   │ 任意表示) │
│  フォルダ    │  │  [フィルターパネル]     │   │         │
│  ツリー      │  └────────────────────────┘   │ ファイル  │
│             │                              │ 情報・    │
│             │  サムネイル一覧 /            │ タグ      │
│             │  画像ビューア /              │         │
│             │  動画プレーヤー              │         │
└─────────────┴──────────────────────────────┴──────────┘
```

- フルスクリーン時はサイドバー・検索バー・右パネルがすべて非表示になる
- 右パネルは `I` キーまたはツールバーボタンで表示切替

---

## 3. 対応ファイル形式

### 画像
`jpg` / `jpeg` / `png` / `webp` / `bmp` / `tiff` / `tif` / `gif` / `heic` / `heif`

### 動画
`mp4` / `mkv` / `avi` / `mov` / `wmv` / `flv` / `webm` / `m4v`

---

## 4. フォルダスキャン

### スキャン範囲
- 選択フォルダ直下 + 1階層下のサブフォルダまでスキャン（最大深さ 2）
- スキャン結果はSQLiteに保存（次回は差分更新）

### スキャン中の表示
- 画面中央にスピナーと「スキャン中…」のオーバーレイを表示
- `Esc` キーでスキャンをキャンセル可能（結果を破棄してギャラリーに戻る）

---

## 5. サムネイル一覧（ギャラリー）

### 表示
- グリッドレイアウトで表示（TanStack Virtual による仮想スクロール）
- サムネイル生成はバックグラウンドで非同期処理
- 生成済みサムネイルはディスクキャッシュに保存（再起動後も再利用）
- 動画ファイルには `▶` バッジを右上に表示

### サムネイルサイズ
- 検索バーのスライダーで 60〜300px の範囲で調整
- 幅が変わると列数が自動調整されて画面いっぱいに並ぶ
- **設定は次回起動時も保持される**（localStorage）

### ファイル情報
- サムネイル下にファイル名・ファイルサイズを表示
- 省略されたファイル名はホバーすると全文をツールチップで表示

### ステータスバー（下部）
| 要素 | 内容 |
|------|------|
| 件数表示 | フィルター適用後の表示件数（フィルターなし時は総件数） |
| 全選択ボタン | 表示中のファイルをすべて選択状態にする |
| 全解除ボタン | すべての選択を解除する |
| 選択件数表示 | 選択中のファイル数と操作ヒント |

---

## 6. ファイル選択

| 操作 | 動作 |
|------|------|
| クリック | 単独選択（ハイライト表示） |
| ダブルクリック | ビューア / プレーヤーで開く |
| `Ctrl` + クリック | 個別にチェック追加 / 解除 |
| `Shift` + クリック | アンカーから指定位置まで範囲チェック |
| `Shift` + `←` / `→` | キーボードで範囲チェック拡張 |
| `Enter` | 選択中のファイルを開く |

- チェック済みファイルは右上に青い ✓ バッジを表示
- アンカー位置は最後に単独クリックした位置

---

## 7. ソート

- **ソート項目**: 名前 / 日付 / サイズ / 種別
- **順序**: 昇順 / 降順
- **設定は次回起動時も保持される**（localStorage）

---

## 8. 検索・フィルター

### 検索バー（バックエンドフィルター）
- ファイル名でのキーワード検索（`Ctrl+F` でフォーカス）
- 全て / 画像のみ / 動画のみ の切り替え

### フィルターパネル（フロントエンドフィルター）
「▼ フィルター」ボタンで展開。アクティブなフィルター数をバッジ表示。

| フィルター | 内容 |
|-----------|------|
| ファイル名 | 部分一致（大文字小文字無視） |
| 種別 | 拡張子単位でトグル選択（`.jpg` `.png` `.webp` `.gif` `.bmp` / `.mp4` `.mkv` `.avi` `.mov` `.wmv` `.webm`）。jpeg は jpg と同一視 |
| 日付 | 更新日時の範囲指定（開始日〜終了日） |
| サイズ | ファイルサイズの範囲指定（KB単位） |
| タグ | Enter で追加、✕で削除（AND条件） |

- 「すべてリセット」ボタンで全フィルター一括解除
- **フィルター適用中にビューアで前後移動すると、フィルター対象ファイルのみを移動**

### 表示オプション
| オプション | 内容 |
|-----------|------|
| 選択のみ | チェック済みファイルのみ表示（チェックが1件以上あるときに表示） |
| 非表示 | サムネイル生成に失敗したファイルを非表示 |

---

## 9. 画像ビューア

### 起動
- ギャラリーでダブルクリック、または `Enter` キー

### 表示・操作

| 機能 | 操作 |
|------|------|
| ズームイン | `+` / `=` キー、または `Ctrl` + ホイール上 |
| ズームアウト | `-` キー、または `Ctrl` + ホイール下 |
| ズームリセット（フィット） | `0` キー、または ⊞ ボタン |
| ドラッグ移動 | マウスドラッグ（ズーム時） |
| 右回転 90° | `R` キー、または ↻ ボタン |
| 左回転 90° | `Shift+R` キー |
| 左右反転 | `H` キー、または ⇆ ボタン |
| 上下反転 | `V` キー |
| 前の画像へ | `←` キー、または ‹ ボタン、またはホイール下 |
| 次の画像へ | `→` キー、または › ボタン、またはホイール上 |
| フルスクリーン | `F` キー、または ⊡ ボタン |
| フルスクリーン解除 | `Esc` キー、または ⊠ ボタン |
| 閉じる | `Esc` キー（非フルスクリーン時）、または ✕ ボタン |
| 情報パネル切替 | `I` キー |

- 画像が変わるたびにズーム・回転・反転はリセット
- ズーム率をパーセント表示（トップバー）

---

## 10. 動画プレーヤー

### 起動
- ギャラリーでダブルクリック、または `Enter` キー

### コントロール

| 機能 | 操作 |
|------|------|
| 再生 / 一時停止 | `Space` キー、または ▶/⏸ ボタン、または画面クリック |
| シーク | プログレスバーをドラッグ |
| 5秒送り | `→` キー |
| 5秒戻し | `←` キー |
| 前のファイルへ | `Shift+←`、または ⏮ ボタン |
| 次のファイルへ | `Shift+→`、または ⏭ ボタン |
| 音量上げ | `↑` キー（+10%） |
| 音量下げ | `↓` キー（-10%） |
| ミュート切替 | `M` キー |
| ループ切替 | `L` キー、または 🔁 ボタン |
| 再生速度 | 0.25x / 0.5x / 1x / 1.5x / 2x（セレクトボックス） |
| フルスクリーン | `F` キー、または ⊡ ボタン |
| 閉じる | `Esc` キー、または ✕ ボタン |

- 再生中はコントロールバーを 2.5 秒後に自動非表示
- マウス移動でコントロールバー再表示
- ループオフ時は最後まで再生すると次のファイルへ自動移動

---

## 11. スライドショー

- 検索バーの「▶ スライドショー」ボタンで開始
- 切り替え間隔は 1〜60 秒で設定可能（デフォルト 3 秒）
- チェック済みファイルがある場合はそのファイルのみを循環
- チェックなしの場合はフォルダ全体を順に表示
- `Space` キーで一時停止 / 再開

---

## 12. プレイリスト

チェックしたファイルの組み合わせを名前をつけて保存する機能。

### 操作
| 操作 | 方法 |
|------|------|
| 保存 | ファイルをチェック後「💾 保存」ボタン → 名前を入力 |
| 読み込み | 「📂 プレイリスト (N)」→ ドロップダウンから名前をクリック |
| 削除 | ドロップダウン内の各プレイリスト名の右にある 🗑 アイコン |
| 選択解除 | ドロップダウン内の「✕ 選択解除」（チェックがある場合に表示） |

- 保存先はブラウザの `localStorage`
- プレイリスト作成時のフォルダパスも保存される
- 別フォルダのプレイリストを読み込むと、自動的にそのフォルダに移動してスキャン後にファイルを選択

---

## 13. タグ管理

### 付与
- 右の情報パネル（`I` キーで表示）のタグ入力欄にタグ名を入力して `Enter` または「追加」ボタン
- 複数ファイルをチェック選択中に追加すると、**全チェックファイルに一括付与**

### 削除
- 情報パネルのタグチップの ✕ ボタン

### 保存先
- SQLite データベース（アプリ再起動後も保持）

---

## 14. ファイル情報パネル

`I` キーまたはツールバーボタンで右パネルに表示。

| セクション | 表示内容 |
|-----------|---------|
| タグ | 付与済みタグ一覧 + 追加・削除 |
| ファイル情報 | サイズ / 更新日時 / 作成日時 / 解像度（画像の場合） |
| 撮影情報（EXIF） | 撮影日時 / カメラメーカー / カメラモデル / レンズ / 焦点距離 / 絞り / シャッター速度 / ISO |
| GPS | 緯度 / 経度 |
| パス | フルファイルパス |

- EXIF・GPS 情報は該当データが存在する場合のみ表示

## 18. パフォーマンス設計

| 項目 | 実装 |
|------|------|
| 大量ファイル対応 | TanStack Virtual による仮想スクロール（表示範囲外はレンダリングしない） |
| サムネイル生成 | Rust バックグラウンドスレッドで非同期処理 |
| サムネイルキャッシュ | 生成済みデータを SQLite に保存して再利用 |
| 画像・動画の読み込み | `readFile` + Blob URL（表示時のみ読み込み） |
| ギャラリーのスクロール位置 | ビューアから戻ったときに元位置を保持（常時マウント + CSS `display` 切替） |
